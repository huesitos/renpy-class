Ren'Py Cheat Sheet

Ren'Py scripting language

init
block of script code for defining all the visual novel/simulation elements like characters, images, variables, positions.
  1. init:
  2. define s = Character("Sam", color="#c8ffc8")
define
defines a character tag with a name and a color in hexadecimal for their name label
image tag identifier
declare an image with a tag and an identifier word
  1. image amy happy = "resources/amy_happy.png"
  2. image background forest = "resources/background_forest.png"
Position
defines a point in screen that can be used to position an image
  1. $ charcenter = Position(xpos=0.5, xanchor=0.5, ypos=0.6, yanchor=0.5)
label
marks the start of a section of the vn. Normally represents a scene. Is common practice to name the first scene as label start
play
reproduces music or a sound effect.
  1. play music - plays a soundtrackplay sound - plays a sound effect
  2. play music "resources/bg_music.mp3"
scene tag identifier
draws a background using the image specified in the identifier
  1. scene bg coderdojochi
show what at where
draws an image at a specific location. what is a tag identifier combination, and at is a Position
  1. show s happy at charcenter
with transition
used in combination with show or scene makes the image appear with the specified transition
  1. scene bg gamecenter
  2. with fade
menu
displays a menu of options for the player to pick. Each option has a different outcome
  1. menu:
  2. "Smash Brothers":
  3. s "I see you like fighting games. That seems like fun. May the best man win!"
  4. "Mario Party":
  5. s "So you like party games! Sounds like we're going to have some good laughs. May the best man win!"
jump
moves from scene to scene (labels)
  1. jump library
return
goes from a scene back to a previous scene where the jump occured. Inside the first label (label start) scene it ends the game

Python code

python
marks the beginning of a block with Python code
  1. python:
  2. gamecenter = False
  3. game = ''
$
marks a line as a Python code
  1. $ game = "Mario Party"
if/elif/else
used to change the flow of the story based on the validity of different elements
  1. if gamecenter:
  2. s "You seemed to enjoy the {i}[game]{/i} game a lot, so I figured you'd like this {b}[gift]{/b}."
  3. else:
  4. s "I saw you were really into that {b}[gift]{/b} book, so I got you a copy from the library bookstore."
